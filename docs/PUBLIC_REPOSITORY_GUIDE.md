# Public Repository Guide

Copyright (c) 2026 Reallexi LLC. All rights reserved.

This repository is the public launcher, contribution, and extension surface for
AI Model Builder. It is not an application source repository or a GitHub Pages
source repository.

## What This Repository Contains

* public installation and validation scripts;
* runtime image and package references;
* contribution, governance, security, support, and policy documents;
* public extension and theme guidance;
* versioned core access metadata.

## What This Repository Must Not Contain

* application or documentation-site source code;
* GitHub Pages configuration or deployment workflows;
* private operating guides or endpoint catalogs;
* internal release instructions or architecture details;
* debug data, databases, credentials, uploads, or artifacts;
* screenshots that expose customer data, local filesystem paths, tokens, or
  private model artifacts.

## Public Guidelines Site

The public-facing site at https://llm.reallexi.io is owned and deployed from
a separately controlled Reallexi LLC repository. GitHub Pages must remain
disabled in this public repository.

## Clean Media Rule

Public screenshots, videos, or animated hero assets must use synthetic sample
content only. They may show high-level concepts such as repository split,
package access, plugin manifests, and community contribution steps. They must
not show credentials, customer prompts, private artifacts, local absolute
paths, or internal source files.
