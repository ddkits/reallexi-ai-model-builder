# Reallexi Private Core Access

This directory contains access metadata, not private source.

Files:

* `reallexi-core-access.json`: readable access manifest.
* `reallexi-core-access.json.gz`: compressed access manifest for package/release pipelines.
* `reallexi-core-access.sha256`: checksum for the compressed manifest.

The runtime core is pulled from private package managers or container
registries. Do not commit private credentials into this public repository.
