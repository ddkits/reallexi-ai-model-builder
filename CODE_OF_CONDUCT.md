# Code of Conduct

AI Model Builder should be a respectful, practical, and useful open-source
community.

## Expected Behavior

* Be respectful and constructive.
* Assume good intent, but accept correction.
* Keep issues and pull requests focused on the project.
* Discuss technical disagreements with evidence and patience.
* Respect maintainer decisions.

## Unacceptable Behavior

* Harassment, threats, insults, or discriminatory language.
* Sharing private information without permission.
* Posting secrets, credentials, customer data, or unsafe exploit instructions.
* Repeatedly demanding unpaid work from maintainers.
* Disruptive comments that prevent useful project discussion.

## Enforcement

Reallexi LLC maintainers may edit, hide, delete, lock, or close content; warn
participants; block users; or restrict participation when needed to protect the
project and community.

Report concerns privately through the security or support channels in this
repository.
